PERSYARATAN SISTEM

Minimum

  • Sistem Operasi: Windows XP/7
  • CPU: 1.6 GHz
  • RAM: 2 GB
  • Kartu Grafis: 256 Mb
  • Ruang Hard Disk: 10 GB

Direkomendasikan

  • Sistem Operasi: Windows 7, 8, 10, 11
  • CPU: 2.5 GHz
  • RAM: 4 GB
  • Kartu Grafis: 512 Mb
  • Ruang Hard Disk: 11 GB

03. INSTALASI GAME

Instal client game di komputer Anda di lokasi yang diinginkan. Tentukan folder tempat Anda menginstal atau mengekstrak arsip, serta beri nama folder tersebut. Jika client didownload dalam bentuk arsip, gunakan WinRAR atau 7zip untuk mengekstrak file.

DOWNLOAD WINRAR DOWNLOAD 7ZIP

04. UPDATE & MAINKAN GAME

Jalankan patcher.exe dari folder yang sudah diekstrak. Patcher akan memeriksa update, dan jika diperlukan, client akan diperbarui secara otomatis. Setelah itu, Anda dapat login ke game.

CARA PATCH MANUAL

1. Ekstrak file patch yang telah kamu download

2. Login Melalui Patcher.exe

3. Setelah update selesai, klik ikon Start untuk mulai bermain.
